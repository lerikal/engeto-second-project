# Projekt: Správa kryptoměnového portfolia

## Popis projektu
Aplikace poskytuje REST API pro správu kryptoměnového portfolia. Umožňuje přidávat kryptoměny, zobrazovat jejich seznam, vyhledávat konkrétní kryptoměnu, aktualizovat její údaje a vypočítat celkovou hodnotu portfolia.

## Použité technologie
- Java 21
- Spring Boot
- Spring Web
- Swagger / OpenAPI
